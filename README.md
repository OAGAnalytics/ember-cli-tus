ember-tus
==============================================================================

An emberized wrapper for the tus.js client: https://github.com/tus/tus-js-client
Based on the tus.io protocol: https://tus.io/


Compatibility
------------------------------------------------------------------------------

* Ember.js v2.18 or above
* Ember CLI v2.13 or above


Installation
------------------------------------------------------------------------------

```
ember install ember-tus
```
